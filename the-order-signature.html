  1. Click Copy signature above.
  2. Open Mail → menu bar → Mail → Settings → Signatures tab.
  3. Select your theorder.global account on the left, then click + to add a signature and name it "The Order".
  4. Important: uncheck "Always match my default message font" (bottom of the window) so the formatting is kept.
  5. Click into the right-hand box, select any placeholder text, and paste (Cmd+V). The lockup and details will appear.
  6. At the bottom, set the Choose Signature dropdown to "The Order" so it's used automatically.
  1. Click Copy signature above.
  2. In Gmail, open Settings (gear icon) → See all settings → General tab.
  3. Scroll to Signature → Create new, name it "The Order".
  4. Click into the signature box and paste (Cmd+V / Ctrl+V) — the lockup and details come through.
  5. Under Signature defaults, set it for new emails (and replies if you want), then Save Changes at the bottom.
  1. Email yourself this signature first: paste it into a message from your Mac, or click Copy signature on the phone if you opened this page there.
  2. On the phone, open the email and copy the signature (tap & hold → Select All → Copy).
  3. Go to Settings → Apps → Mail → Signature (older iOS: Settings → Mail → Signature).
  4. Choose Per Account, tap the box for the theorder.global account, and paste. iOS keeps the logo and formatting when pasted.

The lockup loads from the live website, so it always displays correctly for the people you email. Keep an internet connection the first time the signature is shown.
