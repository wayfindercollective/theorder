Don't paste into iCloud.com web mail — its signature box is plain-text only and strips the logo no matter what. Use the Mail app on Mac/iPhone, or Gmail on the web; both keep the formatting.

To install on your phone (Apple Mail): copy the version you want (or have it emailed to you and copy it from that email), then go to Settings → Apps → Mail → Signature, choose Per Account, tap the theorder.global box, and paste. If it pastes as plain text, shake the phone and tap “Undo Change Attributes” — that restores the logo and formatting.

Gmail: set it once in Gmail on the web (Settings → See all settings → General → Signature → paste → Save). The Gmail iPhone app then uses it automatically, as long as no “Mobile signature” is set in the app.

All images load from the live theorder.global site, so every version displays correctly for the people you email.
